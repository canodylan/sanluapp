# Arquitectura

La arquitectura propuesta se compone de tres componentes:
- Frontend
- Backend
- Base de datos
## Frontend
El frontend será desarrollado utilizando Angular v20.
Utilizaremos Angular para poder crear una interfaz de usuario única que sirva tanto para usuarios Web en ordenador, como para usuarios móviles a través de una AWP.

## Backend
El backend será desarrollado en Java 21, utilizando el framework Spring Boot 3.2.

## Base de datos
La base de datos será PostgreSQL v18

## Contenedores Docker
Cada componente de la arquitectura se ejecutará en su propio contenedor Docker.
